---
title: Traces
description: Inspect compiled context, model actions, tools, retries, and token use.
---

Every model step records:

```ts
type AgentTraceStep = {
  step: number;
  nodeIds: string[];
  prompt: string;
  contextTokens: number;
  rawModelOutput: string;
  action: "tool" | "final" | "invalid";
  tool?: string;
  error?: string;
};
```

Run metadata includes:

- engine version
- tools
- start/completion time
- iteration and token budgets
- model and tool calls
- latest and cumulative input tokens
- output tokens

Use traces to answer:

- which graph nodes did this inference read?
- what exact context did the provider receive?
- what action did the model return?
- which tool ran?
- why did the runtime retry?
- how large was the working context?

Set `traceDir` to persist successful run results as JSON.
