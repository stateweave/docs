---
title: Coding agent example
description: Run a persistent coding agent with the StateWeave causal engine.
---

```ts
import { Agent, AnthropicModel } from "stateweave";

const model = new AnthropicModel({
  apiKey: process.env.ANTHROPIC_API_KEY!,
  model: "claude-3-5-sonnet-latest",
  maxTokens: 4096,
  temperature: 0
});

const agent = new Agent({
  model,
  maxIterations: 30,
  maxPromptTokens: 64_000,
  projectionTargetTokens: 16_000,
  enforceCompletionEvidence: true,
  traceDir: ".stateweave/traces"
});

const result = await agent.run(
  "Inspect the project, fix the token-refresh bug, run the focused check, and report the verified result."
);

console.log(result.finalAnswer);
console.log(result.state);
```

The default workspace tools are available automatically.

Stream engine progress when building an operator UI:

```ts
for await (const event of agent.streamEvents("Add a regression test.")) {
  if (event.type === "progress") {
    console.log(event.progress.phase, event.progress.detail);
    if (event.progress.graph) renderGraph(event.progress.graph);
  }
  if (event.type === "final") console.log(event.result.finalAnswer);
}
```
