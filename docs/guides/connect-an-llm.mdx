---
title: Connect an LLM
description: Implement the provider-neutral model interface.
---

StateWeave does not require a provider SDK. A model adapter implements `complete` and `stream`.

```ts
import type { Model } from "stateweave";

export const model: Model = {
  async complete(input) {
    const response = await fetch("https://provider.example.com/v1/generate", {
      method: "POST",
      headers: { "content-type": "application/json" },
      body: JSON.stringify({ prompt: input.prompt })
    });
    return { text: await response.text() };
  },

  async *stream(input) {
    yield { type: "token", token: (await this.complete(input)).text };
  }
};
```

Then pass it to the single public agent class:

```ts
import { Agent } from "stateweave";

const agent = new Agent({ model });
```

## Anthropic-compatible adapter

```ts
import { Agent, AnthropicModel } from "stateweave";

const model = new AnthropicModel({
  apiKey: process.env.ANTHROPIC_API_KEY!,
  baseUrl: "https://api.anthropic.com",
  model: "claude-3-5-sonnet-latest",
  maxTokens: 4096,
  temperature: 0
});

const agent = new Agent({ model });
```

Provider APIs receive the compiled causal context in `input.prompt`. Keep adapters thin; graph state and projection belong to StateWeave.
