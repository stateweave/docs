---
title: Interactive CLI
description: Inspect compiled causal context, actions, and persistent agent state.
---

Run:

```bash
pnpm cli
```

Then enter tasks:

```txt
stateweave › Remember that the release window is Friday.
stateweave › When is the release window?
```

The CLI keeps one `AgentState` across turns.

## Commands

```txt
/prompt   show the exact compiled causal context
/compare  compare with a traditional transcript
/graph    show a Mermaid graph
/reset    clear agent state
/full     show lossless AgentState JSON
/compact  show compact state
/exit     quit
```

## One-shot examples

```bash
pnpm cli "Inspect the workspace and explain the current API surface." --prompt --graph
```

```bash
pnpm cli "Remember that the project codename is Marigold." --full
```
