---
title: Visualize the graph
description: Render the Agent's causal state as a dependency-free StateGraph view.
---

`AgentState` is lossless persistence. `getGraph()` returns a bounded-text visualization view.

```ts
const result = await agent.run("Inspect the current workspace.");
console.log(result.state); // lossless causal state
console.log(result.graph); // visualization view
```

Render Mermaid with the low-level utility:

```ts
import { graphToMermaid } from "stateweave";

console.log(graphToMermaid(agent.getGraph()));
```

The visualization maps:

- `goal` to `user_input`
- `answer` to `assistant_output`
- tool and resource kinds directly
- causal parents to graph edges

Large payload text is truncated only in the visualization. `AgentState` remains unchanged.

In the CLI:

```txt
/graph
```
