---
title: TypeScript quickstart
description: Create one StateWeave Agent, stream a task, and persist its causal state.
---

## Install

```bash
pnpm add stateweave@dev
```

## Run an agent

```ts
import { Agent, createModelFromEnv } from "stateweave";

const agent = new Agent({
  model: createModelFromEnv()
});

for await (const chunk of agent.stream("Create a tiny HTML todo app in ./todo")) {
  process.stdout.write(chunk);
}

await agent.run("Add keyboard shortcuts.");
console.log(agent.getState());
console.log(agent.getGraph());
```

`Agent` includes workspace-scoped `read_file`, `write_file`, `edit_file`, and read-only `bash_command` tools by default.

## Resume later

```ts
const first = await agent.run("Remember that the release window is Friday.");

const resumed = new Agent({
  model: createModelFromEnv(),
  state: first.state
});

const second = await resumed.run("When is the release window?");
```

Imported state is validated before use.

## Inspect the engine

```ts
for await (const event of agent.streamEvents("Inspect the workspace")) {
  if (event.type === "progress") {
    console.log(event.progress.phase, event.progress.detail);
    console.log(event.progress.prompt);
    console.log(event.progress.graph);
  }
}
```

Use `stream()` for answer text and `streamEvents()` only when you need engine diagnostics.
