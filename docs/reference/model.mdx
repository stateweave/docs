---
title: Model interface
description: The provider adapter contract used by Agent.
---

```ts
export type Model = {
  complete(input: ModelInput): Promise<ModelOutput>;
  stream(input: ModelInput): AsyncIterable<ModelToken>;
};
```

```ts
export type ModelInput = {
  prompt: string;
  mode?: "graph_ops" | "text";
  parameters?: ModelParameters;
  system?: string;
  signal?: AbortSignal;
};
```

The public `Agent` calls the model in text mode. `prompt` contains the compiled causal working context; `system` contains the provider-level action instruction.

```ts
export type ModelOutput = {
  text: string;
  usage?: {
    inputTokens: number;
    outputTokens: number;
    totalTokens: number;
  };
};
```

Usage data feeds run metrics when the provider supplies it; otherwise StateWeave estimates tokens locally.
