---
title: Tool interface
description: The Zod-validated action contract.
---

```ts
import type { z } from "zod";

export type Tool = {
  name: string;
  description: string;
  schema: z.ZodSchema;
  execute: (args: unknown) => Promise<unknown>;
};
```

Tool descriptions are included in the system contract. The model requests one tool through a `TOOL_CALL` envelope.

StateWeave validates arguments, executes the tool, records a `tool_call` node and child `tool_result`, then extracts resource versions from path/hash/mutation metadata when available.

A failed tool result remains visible to later inference and does not count as successful completion evidence.
